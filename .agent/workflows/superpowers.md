# Superpowers Development Workflow

Follow this process for all significant development tasks.

1. **Brainstorming & Design**
   Before writing any code, explore requirements and design.
   - Use `brainstorming` skill

2. **Setup Workspace**
   Create an isolated environment for the task.
   - Use `using-git-worktrees` skill

3. **Implementation Planning**
   Break down the work into detailed steps.
   - Use `writing-plans` skill

4. **Execution**
   Execute the plan using subagents or batch execution.
   - Use `subagent-driven-development` OR `executing-plans` skill

5. **Test-Driven Development**
   For every code change, follow Red-Green-Refactor.
   - Use `test-driven-development` skill

6. **Review**
   Verify quality before completion.
   - Use `requesting-code-review` skill

7. **Completion**
   Merge changes and cleanup.
   - Use `finishing-a-development-branch` skill
